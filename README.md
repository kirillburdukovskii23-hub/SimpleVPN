# SimpleVPN

Надоело что ВПН на ПК платные? Надоело, что нормального ВПН на ПК нету?
SimpleVPN - название говорит за себя, простой ВПН.
Надежность, простота, доступность - это наше

Ну а самое главное, что он работает!

ПРИМЕЧАНИЯ

Не пугайтесь консоли, не выключайте ее пока ВПН не запуститься, и не выключайте консоль при работе ВПН.
